# N-Queens
The N-Queens Solver is a Java program that finds solutions to the N-Queens problem. The N-Queens problem is a classic puzzle where you need to place N chess queens on an NÃ—N chessboard in such a way that no two queens threaten each other.

This program provides a solution for solving the N-Queens problem using a Steep Hill and Genetic Algorithm. It explores different combinations of queen placements on the chessboard to find valid solutions.
